# Dev Voice (Universal) 🔊

Make your coding experience more immersive with premium sound effects for every event. **Dev Voice** is built for modern AI-powered coding and is fully compatible with **VS Code, Cursor, Antigravity, and other VS Code forks.**

## 🚀 Universal Compatibility
Dev Voice uses the universal VS Code Extension API, making it compatible with:
- **VS Code**: Traditional coding with tactile feedback.
- **Cursor**: Enhanced AI coding with custom sound triggers.
- **Antigravity**: Native support for AI-driven development workflows.
- **Windsurf / PearAI**: Smooth integration with any VS Code-based environment.

## ✨ Features
- **Smart Logic**: Distinct sounds for typing, saving, and deleting.
- **AI Error Awareness**: Variable meme sounds based on how many files have errors.
- **Git Awareness**: Alerts for branch switching with uncommitted changes or merge conflicts.
- **Dynamic Dashboard**: Full control panel to toggle and preview sounds.

## 🛠️ Installation on Different IDEs

### VS Code & Cursor
1. Open the editor.
2. Go to the Extensions view (`Cmd+Shift+X`).
3. If you are developing locally, press `F5` in the source folder to launch.

### Antigravity & AI Agents
As an AI-powered extension, Dev Voice provides auditory cues that help you keep track of what your AI agent is doing in the background.

## 🎛️ Control Panel
Run the command `Dev Voice: Open Dashboard` to open the interactive settings. Here you can:
- Toggle individual sound triggers.
- Preview every sound in the library.
- Customize your auditory experience.
